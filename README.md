# Karmell

Karmelli koduleht, mida saab deployda Vercelis otse selle repo juurkaustast.

## Deploy

Vercelis vali selle repo `Root Directory`-ks projekti juurkaust ja Framework Presetiks `Other`.

Kuna leht on staatiline, piisab sellest, et avaleht on failis `index.html` ning pildid/asendid jäävad samadesse suhtelistesse kaustadesse.
